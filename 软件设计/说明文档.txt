RBAC权限监控管理系统软件说明文档
程序所在目录：项目代码
前端目录：项目代码/web
后端目录：项目代码/server
前端部署
安装Node.js：
下载并安装Node.js，确保版本符合项目要求。
安装依赖：
进入项目目录，运行 npm install 或 yarn install 安装项目依赖。
构建项目：
运行 npm run build 或 yarn build 构建生产环境的前端代码。
部署静态文件：
将构建生成的 build 目录中的静态文件部署到Web服务器（如Nginx）。
后端部署
安装Go：
下载并安装Go语言环境，确保版本符合项目要求。
安装依赖：
进入项目目录，运行 go mod download 安装项目依赖。
编译项目：
运行 go build 编译项目，生成可执行文件。
配置环境：
配置数据库连接信息、API密钥等环境变量。
启动服务：
运行编译后的可执行文件，启动后端服务。
